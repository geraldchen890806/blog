## Hello, New World

This is a brand-new start for my tech blog.

Back in 2015 I built the first version with React + Webpack and wrote plenty of technical notes on it — from ES6 Generators to PWAs, from Webpack 4 to all kinds of frontend concepts. That project stayed with me for many years.

Time for something new.

## Why Astro

Why I picked [Astro](https://astro.build/):

1. A blog doesn't need a client-side JS runtime — pure static HTML loads fast
2. Writing a post is just writing a `.md` file
3. TypeScript, Tailwind CSS, and automatic OG image generation are all built in
4. Zero JS by default; add interactivity only where you need it

Compared with the old React + Webpack setup, page size dropped by an order of magnitude. First paint is noticeably faster — the kind you can see with the naked eye.
