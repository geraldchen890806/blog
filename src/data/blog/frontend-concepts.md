## HTTP vs HTTPS

- HTTP：明文传输，运行在 TCP 之上
- HTTPS：SSL/TLS 加密，添加了加密和认证机制
- HTTP/2：多路复用，一个 TCP 连接传输所有请求

## 实用 CSS 技巧

1. `attr()` 抓取 `data-*` 属性
2. `position: sticky` 粘性定位
3. `text-align-last` 对齐两端文本
4. `writing-mode` 排版竖文
5. `object-fit` 规定图像尺寸
6. `transform` 描绘 1px 边框
7. `filter: grayscale(1)` 开启悼念模式
8. `:valid` 和 `:invalid` 校验表单
9. `:focus-within` 分发冒泡响应
