## 你好，新世界

这是我的技术博客的全新开始。

从 2015 年开始，我就在用 React + Webpack 搭建的博客记录技术笔记。那个项目陪伴了我很多年，从 ES6 Generator 到 PWA，从 Webpack 4 到各种前端概念，承载了不少学习和成长的痕迹。

如今，是时候给它一次"重生"了。

## 为什么选择 Astro

选择 [Astro](https://astro.build/) 的理由很简单：

1. **静态优先** — 博客不需要客户端 JavaScript 运行时，纯静态 HTML 加载极快
2. **Markdown 原生支持** — 写文章就是写 `.md` 文件，简单直接
3. **现代工具链** — TypeScript、Tailwind CSS、自动 OG 图片生成，开箱即用
4. **零 JS by default** — 只在需要的地方引入交互，性能天然优秀

相比之前的 React + Webpack 方案，Astro 生成的页面体积小了一个数量级，首屏加载速度质的飞跃。
