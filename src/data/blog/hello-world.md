## 你好，新世界

这是我的技术博客的全新开始。

2015 年用 React + Webpack 搭了第一版博客，写了不少技术笔记。从 ES6 Generator 到 PWA，从 Webpack 4 到各种前端概念，那个项目跟了我很多年。

现在换个新的。

## 为什么选择 Astro

选 [Astro](https://astro.build/) 的理由：

1. 博客不需要客户端 JS 运行时，纯静态 HTML 加载快
2. 写文章就是写 `.md` 文件
3. TypeScript、Tailwind CSS、自动 OG 图片生成都有
4. 默认零 JS，需要交互的地方再加

比原来的 React + Webpack 方案，页面体积小了一个数量级。首屏加载快了很多，肉眼可感的那种。
